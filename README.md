fmmtl
=====

FMM Template Library

Primary Authors:
 Cris Cecka (ccecka@seas.harvard.edu)
 Simon Layton

Contributors:
 Christopher Cooper
 Aparna Chandramowlishwaran


Dependencies:
 g++ 4.7 or higher
 Boost library headers (Tested with version 1.53)
Optional:
 CUDA 5.0 and Thrust


Building:
 Enter unit_tests/ or examples/
 Edit Makefile.inc to provide dependency paths
 'make'
